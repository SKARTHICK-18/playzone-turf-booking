# ⚽ PlayZone

A React-based Turf Booking Application.

## Features

- Browse Turfs
- Search & Filter
- Turf Booking
- Time Slot Selection
- Ground Selection
- Booking Summary
- Booking Confirmation Modal
- My Bookings
- Responsive UI

## Tech Stack

- React.js
- React Router
- CSS
- JavaScript
- LocalStorage

## Installation

```bash
npm install
npm run dev
```
